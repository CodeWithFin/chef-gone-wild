# Assets Folder

This folder contains images and other static assets for the Chef Gone Wild application.

## Carousel Images

Place the following images in this folder for the homepage carousel:

1. **carousel-1.jpg** - Cocktail image (first slide)
2. **carousel-2.jpg** - Food/Chicken image (second slide)  
3. **carousel-3.jpg** - Bar Interior image (third slide)

### Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1920x1080 pixels or larger
- **Aspect Ratio**: Landscape (16:9 or similar)
- **File Size**: Optimize for web (under 500KB per image recommended)

### How to Add Images

1. Download or prepare your images
2. Rename them to `carousel-1.jpg`, `carousel-2.jpg`, and `carousel-3.jpg`
3. Place them in this `public/assets/` folder
4. The carousel will automatically use them on the next page load

### Fallback

If the local images are not found, the carousel will automatically fall back to external URLs. This ensures the site continues to work even if images haven't been added yet.
